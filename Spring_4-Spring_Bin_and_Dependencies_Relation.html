Spring [4] - Spring Bin and Dependencies Relation

Spring [4] - Spring Bin and Dependencies Relation

Register Srping Bin and Set Dependencies Relation

  • Set dependcies relation for member controller to use member service and member repository

Add dependencies relations in member controller

package hello.hellospring.controller;

import hello.hellospring.service.MemberService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Controller;

@Controller
public class MemberController {

    private final MemberService memberService;

    @Autowired
    public MemberController(MemberService memberService) {
        this.memberService = memberService;
    }
}
  • If constructor has @Autowird, Spring container finds the related object. This method of dependencies relation outside is called DI(Dependency Injection)
  • Prior to this test, the developer had to put it in manually, but spring does it for us due to @Autowired

Error occur

Consider defining a bean of type 'hello.hellospring.services.MemberService' in your configuration.
  • There are 2 ways to register Spring bin
    • Component scan and dependencies relation auto set
    • Register Spring bin using Java code

Component Scan and Dependencies Relation Auto Set

  • @Component: Spring bin is automatically registered if annotation exists
  • @Controller: Controller got automatically registered to Spring bin due to component scan
  • The following annotation including @Component get automatically registered to Spring bin:
    • @Controller
    • @Service
    • @Repository

Member Service Spring Bin Register

@Service
public class MemberService {

    private final MemberRepository memberRepository;

    @Autowired
    public MemberService(MemberRepository memberRepository) {
        this.memberRepository = memberRepository;
    }

}

Tip

  • If constructor uses @Autowired, Srping bin will be found to be injected at the time the object is created

Member Repository Spring Bin Register

@Repository
public class MemoryMemberRepository implements MemberRepository {}
  • memberService and memberRepository are registered to Spring bin

Tip

  • Spring registers in single tone by default when registering Spring Continaer and Spring Bin.

Register Spring Bin Using Java Code

  • Remove @Serivce, @Repository annotation and continue
package hello.hellospring;

import hello.hellospring.repository.MemberRepository;
import hello.hellospring.repository.MemoryMemberRepository;
import hello.hellospring.service.MemberService;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class SpringConfig {

    @Bean
    public MemberService memberService() {
        return new MemberService(memberRepository());
    }

    @Bean
    public MemberRepository memberRepository() {
        return new MemoryMemberRepository();
    }
}
