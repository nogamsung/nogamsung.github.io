Spring Boot - BaseTimeEntity for SQL

Use @CreatedDate and @LastModifiedDate

Use @CreatedDate and @LastModifiedDate to avoid the lines of code for implementing createdDate and lastModifiedDate.

1️⃣ Create BaseTimeEntity.java

@Getter
@MappedSuperclass
@EntityListeners(AuditingEntityListener.class)
public abstract class BaseTimeEntity {
    @CreatedDate // Manage created date
    private LocalDateTime createdDate;
    @LastModifiedDate // Manage last modified date
    private LocalDateTime modifiedDate;
}

2️⃣ Enable JPA Auditing in Application.java

@EnableJpaAuditing
@SpringBootApplication
public class Application {

	public static void main(String[] args) {
		SpringApplication.run(Application.class, args);
	}
}

Use the above BaseTimeEntity class and extend it to the Entity classes, it will automatically manage createdDate and lastModifiedDate.
