Unreal Engine - Game Mode and Pawn

Concept of Class

  • What is Class?
    • C++: expressed in object as it is an object-oriented language
    • Object is implemented in class
    • Actor is a part that is implemented in class
  • What is Blueprint?
    • In Unreal Engine, you can use blueprint, similar to C++, it is a visual script language
    • The blueprint is a C++ class internally
    • Therefore, C++ class and blueprint class is similar in characteristic
  • Ways to implement class: C++ or Blueprint
    • The way it to implement is not essential
    • Categorized into C++ class or blueprint class
  • Example of Class
    • Object class, Actor class, pawn class, and so on

Game Framework

  • Every game needs to have basic structure
    • Player character, player controller, camera, and so on
    • Even if you use an empty template, you need include a minimum game structure
    • The game structure in UE is “Game Framework”
  • Game Mode
    • Object that express the rules for a game to be played
    • Decides the player and camera in a map during the “Game Mode”
    • If it is not altered, the standard game mode is set to default

Core Class

  • Core classes in UE
    • Object: A class located at the top of all class, all of the classes are derived from this class.
    • Actor: Place a class in level. It is a child class of an object.
    • Pawn: A class that has a physical looks, such as player or A.I. It is child class of an actor.
    • Character: A class that is able to walk or do any type of movement function. It is a child class of a pawn.
    • Player controller: A class that controls a player’s pawn. It is a child class of an actor.
    • Game mode: A class that express the rules for a game. It is a child class of an actor.
  • Name of the core class
    • Class is usually omitted. So just say object, actor, pawn, player controller, and character.
    • Name of the class implemented in C++: UObject, APawn, ACharacter, APlayerController, AGameMode

Blueprint

  • What is blueprint?
    • Visual script system provided by UE
      • Node based development environment that uses the mouse to connect commands.
      • Easy. -> All producers can participate in development.
      • Block assemble form that reduces the grammatical error.
    • Good when making interactive assets
      • Event process script
    • Easy to combine various functions
  • Making blueprint object
    • You can use C++ to make a class or use the blueprint.
    • Blueprint class is a powerful means

Concept of Component

  • Component
    • A single blueprint class has many component inside a component container
    • Component is added to a blueprint class to act as a class
    • Different components do different tasks
  • Types of component
    • Actor component: A base class component that does not have any transform data.
    • Scene Component: A child class of an actor component that has transform data.
    • Primitive component: A child class of a scene component that has graphic expression.
